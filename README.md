# Harris-ShutterTool

Harris-ShutterTool is a static web application to generate Harris Shutter composites from three source images with channel assignment in RGB or CYM mode.

It lets you:

- load 3 images,
- assign each image to one channel,
- switch between RGB and CYM fusion,
- generate the fused output,
- download the final PNG.

Web version:

- https://enheragu.github.io/Harris-ShutterTool/

Source gist with python version of Harris-Shutter fusion algorithm:

- https://gist.github.com/enheragu/4e3df5fcf7eb60f12ca76c28b74a0905
